What I do

Front-End

I implement web designs through coding using HTML5, CSS3, Bootstrap/SASS and Javascript. I am currently learning ReactJS to build reusable User Interface. I like to develop web pages, think about the design and how to optimize the code. My goal is to build beautiful, responsive, ergonomic and interactive websites.

Back-End

I use PHP and Symfony 4 to make dynamic and complex websites. I build databases with MySQL. I also have knowledge in Ruby on Rails and PostgreSQL.

UI/UX Design

When designing websites, I think about the user experience : how the product can meet the user’s needs? What functionalities are important? Why the user will need that product? I love to create new designs or reinvinting existing ones. I use Figma to prototype but also have Sketch or InVision.
